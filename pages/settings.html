    App Settings

    APP BEHAVIOR
    Open VALTracker automatially right when your PC starts!
    CLOSE BUTTON
    Closing the app window will only minimize it to your system Tray.
    PERFORMANCE
    Enable or disable the App's Hardware Acceleration. Requires a restart for changes to apply.

    Theme Settings

    THEMES
    Select a Theme you like!
    CUSTOM THEMES
    Edit or delete the Custom Theme that you are currently using.
    Create a custom Theme using the in-app Theme Creator!
    Import a Theme for VALTracker that you've downloaded!
    Open the Themes Folder with the Button on the right and put the Theme File in there.

    Rich Presence Settings

    RICH PRESENCES
    Show your friends that you're using VALTracker by Activating this setting!
    Let Discord show a more detailed Rich Presence about your current VALORANT Match!
    (This includes the Map, the current Score, the Agent you're playing, Time elapsed since and the Mode you're playing.)

    Riot Account Settings

    ACCOUNT MANAGEMENT
    Open a login window to add an account!
    Log out from all your Accounts. If you want to just remove one account, open the "Switch Account" window from the Navslide.

    Other Settings

    HUB DISPLAY NAME
    Change the Username that is displayed at the top of your Hub!
    Change your Hub Username back to your Playername.
    DANGER ZONE
    Delete all of your data stored on your PC and restart VALTracker to re-do the setup process.
    Only do this if something isnt working properly!
